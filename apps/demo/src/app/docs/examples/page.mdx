# Examples

Two useful starting points:

## 1. Minimal package wiring

Use a short config, one or two seed steps, and a simple server route that returns `engine.generateStep(session)`.

## 2. AI readiness audit

The public demo uses an AI readiness audit config with:

- deterministic opening steps
- multiple scoring dimensions
- a real API route at `/api/form`
- optional model injection through `OPENAI_API_KEY`
